# 音乐乐谱管理系统用户手册

## 软件简介

音乐乐谱管理系统是一款专为音乐爱好者、演奏者、教师和学生设计的乐谱管理软件。它帮助您系统化地整理、分类和查看各类乐谱文件，支持PDF和多种图片格式，提供便捷的浏览和管理功能。

## 安装与启动

### 首次启动时：

1. 启动软件，系统会提示选择乐谱库存储位置
2. 建议选择具有足够空间的磁盘目录
3. 系统会自动创建数据库和配置文件夹

### 后续启动：

直接运行程序，自动加载上次的配置

## 界面介绍

### 3.1 界面布局

（图略）

### 3.2 功能区说明

- **左侧面板**：按不同维度（全部、分类、作曲家、调性、名称）分类显示乐谱
- **右侧网格**：以缩略图形式展示乐谱，支持滚动浏览
- **工具栏**：常用功能快捷按钮

## 基本管理操作指南

### 4.1 导入新乐谱

1. 点击菜单栏【文件】->【导入乐谱...】
2. 在导入窗口中填写乐谱信息：
   - 乐谱名称：必填，如“月光奏鸣曲”
   - 作曲家：如“贝多芬”
   - 调性：从下拉列表选择，如“C大调”
   - 分类：选择已有分类或“无”
   - 版本/类型：可填写版本号或类型描述
3. 选择文件：
   - **PDF文件**：单文件选择
   - **图片文件**：多文件选择（支持PNG、JPG、BMP）
   - **若想要删除文件，可在文件列表中双击，即可删除对应文件**
4. 点击【OK】完成导入

### 4.2 浏览乐谱

#### 4.2.1 按分类浏览
- 在左侧下拉框选择“按分类”
- 展开分类树，点击具体分类
- 右侧网格显示该分类下所有乐谱

#### 4.2.2 按作者浏览
- 选择“按作曲家”
- 点击作曲家名字
- 查看该作曲家的所有作品

#### 4.2.3 按名称浏览
- 在“按名称”模式下，按首字母分组查找

#### 4.2.4 按调性浏览
- 在“按调性”模式下，按调性分类查找

### 4.3 查看乐谱详情

**方法一**：双击乐谱缩略图  
**方法二**：右键点击乐谱 → 选择“打开”

### 4.4 编辑乐谱信息

1. 点击菜单栏【管理】->【乐谱管理】
2. 在左侧列表选择要编辑的乐谱
3. 修改相关信息：
   - 可修改名称、作曲家、调性等基本信息
   - 可修改所属分类
   - 可更换关联文件
   - 可更新备注信息
4. 点击【保存修改】

### 4.5 删除乐谱

1. 在【乐谱管理】界面选择乐谱
2. 点击【删除乐谱】
3. 确认删除（注意：会同时删除物理文件）

## 乐谱查看器功能

### 6.1 打开查看器

- 从主界面双击乐谱打开
- 最大化显示，左侧为信息，右侧为乐谱内容，下方为操作按钮

### 6.2 基本操作

快捷键一览：

- ← 左箭头 ：上一页
- → 右箭头 ：下一页
- Ctrl + 滚轮 ：缩放乐谱
- Ctrl + +（=） ：放大
- Ctrl + - ：缩小
- Ctrl + 0 ：恢复原始大小
- 鼠标拖拽 ：移动乐谱位置

### 6.3 多页乐谱

- **PDF文件**：自动分页，使用箭头翻页
- **图片文件**：每个图片为一页，按顺序排列

### 6.4 节拍器功能

1. 点击下方节拍器按钮或通过主界面工具栏打开
2. 设置速度和节拍类型
3. 点击播放/暂停

## 分类管理

### 7.1 创建新分类

1. 进入【管理】->【分类管理】
2. 点击【新建分类】
3. 选中新创建的分类并输入分类名称和备注
4. 点击【保存】

### 7.2 编辑分类

1. 在分类列表中选择要编辑的分类
2. 修改名称或备注
3. 保存更改

### 7.3 删除分类

点击【删除分类】按钮即可  
注意：删除分类不会删除乐谱，但相关乐谱的分类信息会被清空

## 实用技巧

### 7.1 批量导入

对于多页图片乐谱，可一次性选择所有页面图片，系统会按文件选择顺序排序。

### 7.2 快速刷新

通常情况下程序会在需要时自动刷新界面，但若出现了未同步的情况，可以按 Ctrl+R 手动刷新乐谱网格，解决偶尔显示不同步的问题。

### 7.3 文件组织

所有乐谱文件自动保存在：【配置目录】/【乐谱ID】/ 下，可手动备份整个配置目录。

## 注意事项

如遇到未知错误，请联系开发者。

### 8.1 文件安全

删除乐谱时会同时删除物理文件，请谨慎操作。

### 8.2 格式支持

PDF需完整版本，图片支持PNG、JPG、BMP格式。

### 8.3 文件大小

建议单文件不超过50MB以保证流畅性。

### 8.4 定期备份

重要乐谱建议定期备份整个乐谱库目录。

## 获取帮助

- 软件内点击【帮助】->【用户手册】查看本文档
- 【帮助】->【关于】查看版本信息
- 遇到问题可记录下操作步骤和错误提示，便于排查
